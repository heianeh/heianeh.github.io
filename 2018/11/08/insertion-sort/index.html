INSERTION SORT

Posted on 2018-11-08 | In sort algorithm

Basic Idea

Insertion sort is a simple sorting algorithm. Assuming that we get one digit at a time instead of getting all numbers at first. For me, I would insert the new one to the existing sorted list once I receive it. So the basic idea of insertion sort is that insert the next elements into the sorted list one by one.

Analysis

As discussed in the Basic Idea, we receive one element once, so that we need to receive n times (Actually n - 1 times, the first elements can be ignored.) (line 5).

For each element, we should find the proper index in the numbers list (line 6-10). Because we have to insert the new element into the front list, so all the elements which are greater than the new elements have to move back one step, line 10 works for these.

Line 11: insert the new elements into the right position.

Example

Example list of numbers:

[5, 9, 2, 4, 7, 1]

For the first receive:

  1. i = 1
    newNum = 9
    j = 1
    numbers[j - 1] = 5
    numbers [5, 9, 2, 4, 7, 1]

But numbers[0] < newNum, so the numbers list:

[5, 9, 2, 4, 7, 1]

  1. i = 2
    newNum = 2
    j = 2
    number[j - 1] = 9

newNum < number[j - 1] so that j = j - 1 = 1, numbers[j] = numbers[j - 1]. Then

  1. i = 2
    newNum = 2
    j = 1
    number[j - 1] = 5
    numbers [5, 9, 9, 4, 7, 1]

newNum < number[j - 1] so that j = j - 1 = 0, numbers[j] = numbers[j - 1]. Then

  1. i = 2
    newNum = 2
    j = 0
    number[j - 1] = out of bounds
    numbers [5, 5, 9, 4, 7, 1]

now j is smaller than 1 so the numbers list:

[2, 5, 9, 4, 7, 1]

Continue to do the same process and the numbers will change as following:

[2, 4, 5, 9, 7, 1]
[2, 4, 5, 7, 9, 1]
[1, 2, 4, 5, 7, 9]
Finished

Time Complexity

We do n receive, for each receive, we do 1 to (n - 1) comparisons to find the proper position.
So for the best situation (the list is already sorted as we want), the time complexity is O(n)
For the worest case(the list is reversed as the final result), the time complexity is O(n^2)
For average case, do n / 2 comparisons for each receive, so the time complexity is O(n^2)
Obviously, the space complexity is constant. T(1)
