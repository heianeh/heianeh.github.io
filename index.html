Basic Idea

Insertion sort is a simple sorting algorithm. Assuming that we get one digit at a time instead of getting all numbers at first. For me, I would insert the new one to the existing sorted list once I receive it. So the basic idea of insertion sort is that insert the next elements into the sorted list one by one.
